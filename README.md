A SIMPLE REGEX MATCHER: ONLY ALLOWS "|" "." "\*" OPERATIONS AND NO \ BASED ESCAPE CHAR:
SO MATCHES LIKE A*B  A*B|C (A|B)*B ETC

WHAT IT DOES...
REGEX-> PREFIX -> NFA
WE DO BFS ON NFA GRAPH AND CHECK IF THE FINAL SET OF NODES HAS FINAL STATE OR NOT
